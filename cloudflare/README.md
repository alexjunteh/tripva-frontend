# Tripva — Cloudflare OG Worker

Dynamic `<meta og:*>` rewriting for shared trip links. Social crawlers see a
personalized preview card (destination, dates, hero photo) generated by the
Vercel `/api/og?id=<gist_id>` endpoint.

Regular browsers pass through untouched for lowest latency.

## Architecture

```
Crawler (FB/Twitter/Slack/etc.) → tripva.app/trip?id=X
    └─ CF Worker og-worker.js detects UA → fetches trip from /api/trip
       → HTMLRewriter patches meta tags in trip.html → returns modified HTML
       └─ og:image points to https://tripai-backend.vercel.app/api/og?id=X
          └─ Vercel Edge function renders 1200x630 PNG with @vercel/og

Real user → tripva.app/trip?id=X
    └─ Worker detects non-bot → passes through untouched
       └─ trip.html loads as before, hydrates from localStorage/backend
```

## Deploy

### One-time setup (local)

```bash
# In tripva-frontend repo
cd cloudflare
npm install -g wrangler
wrangler login         # opens browser, sign in with the Cloudflare account
                       # that owns tripva.app
```

### Deploy

```bash
wrangler deploy
```

Wrangler reads `wrangler.toml`, uploads `og-worker.js`, and activates the two
routes (`tripva.app/trip` + `tripva.app/trip.html`). Deploy is near-instant.

### Verify

```bash
# Crawler UA → should see dynamic meta tags
curl -sH "User-Agent: facebookexternalhit/1.1" \
  "https://tripva.app/trip?id=SOME_GIST_ID" | grep "og:image"

# Regular UA → should see static meta tags (unchanged)
curl -s "https://tripva.app/trip?id=SOME_GIST_ID" | grep "og:image"

# Manual override for testing — ?og=1 forces the rewrite even from a browser
curl -s "https://tripva.app/trip?id=SOME_GIST_ID&og=1" | grep "og:image"
```

Then test a real share at https://www.opengraph.xyz/ by pasting a trip URL.

## Changing the OG endpoint URL

Edit `og-worker.js` → `OG_ENDPOINT` constant, then `wrangler deploy` again.

## Rollback

```bash
wrangler delete
```

Routes go away, trip.html serves directly from CF Pages with its static meta
tags — exactly how it works today without the worker.
